# MarkerMap

Projekt realizowany "po godzinach", nazwa robocza MakerMap - w zamyśle finalnie FintIt.

## Technologie

- Android (Kotlin),
- ViewModel,
- Coil,
- Pager,
- Dagger Hilt,
- Google Maps,
- Coroutine,
- Firebase Auth,
- Firebase Realtime Database,
- Firebase Storage.
